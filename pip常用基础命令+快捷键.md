### pip常用命令

#### 1.查看已经安装的第三方库：

pip list

#### 2.直接安装库：

pip install 库名

#### 3.指定版本安装：

pip install robotframework == 2.8.7

#### 4.卸载已安装的库：

pip uninstall 库名

#### 5.导出安装的列表：

pip freeze > 库名

#### 6.从名.txt文件安装库：

pip install -r 名.txt

#### 7.查看pip版本：

pip -version

#### 8.升级pip:

pip install -- upgrade pip

#### 9.查看要升级的库：

pip list -o

#### 10.验证已安装的库是否兼容：

pip check packag-name
