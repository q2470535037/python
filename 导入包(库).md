### 导入包(库)

在 python 用 import 或者 from…import 来导入相应的模块

#### 将整个模块(somemodule)导入

import somemodule

#### 从某个模块中导入某个函数

from somemodule import somefunction

#### 从某个模块中导入多个函数

from somemodule import firstfunc, secondfunc, thirdfunc

#### 将某个模块中的全部函数导入

from somemodule import *

#### 将某个模块改名(改为s)

import somemodule as s
